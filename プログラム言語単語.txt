テスタビリティ：可测性
ステートメント：声明、宣言
プライマリコンストラクター：一時的なコンストラクター
First Class：该类型的值可以作为函数的参数和返回值，也可以赋给变量。
Second Class：该类型的值可以作为函数的参数，但不能从函数返回，也不能赋给变量。
Third Class：该类型的值作为函数参数也不行
リファクタリング：代码重构
イディオム：习语，惯用语
トラフィック：访问量
高階関数：関数に関数を渡す
インクリメント：整数型の変数の値を1増やす処理
デクリメント：整数型の変数の値を1減じる処理

糖衣構文：プログラミング言語において、読み書きのしやすさのために導入される構文であり、既に定義されている他の構文の（人間にとってより理解しやすい）書換えとして定義されるもののことである。
例：2 + 1 => 2.+(1)

シグネチャ：オブジェクト定義の時、＝符号の左側
定義：オブジェクト定義の時、＝符号の右側
例：def abs(n: Int): Int = {
      if (n < 0) -n
      else n
    }
